3.染色效果

filter: sepia() saturate(4) hue-rotate(295deg)。

可以调节不同的值得到不同的结果。

sepia() 将图像转换成深褐色，默认值伪0，值在0% -100% 之间，定义转换的比例，0表示无变化，100%表示完全是深褐色。

saturate() 转换图像的饱和度，0表示完全不饱和。100% 则图像无变化，默认值为1。

hue-rotate()该图像定义色相旋转，超过360deg，相当于又绕了一圈，0deg是默认值，表示图像无变化，表示图像会被调整的色环角度值。

好像兼容性不好，没有显示出效果----

hsl 和 hsla。h----色调，s----饱和度，l-----亮度，a---- 透明度

background-blend-mode:定义了背景层的混合模式，取值：
                luminosity : 亮度模式，会保存上层元素的hsl亮度信息，并哦哦那个它的下层西区色相和饱和度信息
                multiply : 正片叠底模式，
                screen : 滤色模式，
                overlay : 叠加模式，
                darken : 变暗模式，
                lighten : 变亮模式，
                color-dodge : 颜色减淡模式，
                saturation : 饱和度模式，
                color : 颜色模式
