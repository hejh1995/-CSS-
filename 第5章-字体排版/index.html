7.现实中的文字效果

(1)凸版印刷效果

只要文字不是黑色，背景不是纯黑或纯白就可以使用这种特效。浅色背景上使用深色字体是，在文字底部加上浅色投影

凸显效果主要是由色差、text-shadow体现的。text-shadow:x轴位移，y轴位移，阴影模糊半径，颜色。深色背景使用浅色字体，在文字顶部加深色投影

(2)空心字效果

使用text-shadow实现,多重投影.描边宽度大时会很奇怪，比如3px时

svg ，stroke ：定义一条直线，stroke-linejoin：描边转角的表现形式（miter-直角，round-圆角，bevel-拐一下的直角）

(3)文字外发光效果

光，text-shadow: 0 0 .1em, 0 0 .3em;

(4)文字凸起效果

使用一长串的累加的投影，不设置模糊，并以1px的跨度逐渐错开（y轴），使颜色逐渐变暗，然后在底部加一层强烈模糊的暗投影，从而模拟完全的立体效果

使用一长串的累加的投影，不设置模糊，并以1px的跨度逐渐错开（x轴，y轴），颜色始终为黑色
