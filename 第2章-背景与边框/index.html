2.多重边框

box-shadow 不占用布局空间，不影响鼠标事件。 box-shadow: 0 0 0 10px #ffdcee, 0 0 0 15px #fbaaff, 0 2px 5px 30px rgba(0,0,0,0.6);

outline 也不占用布局空间，只适合两层边框的。border: 20px solid #fdfff0; outline: 15px solid #ffdcee;

3.灵活的背景定位

background-position: right 20px bottom 10px

padding: 10px; /// packground-origin ：conten-box

background-position: calc(100% - 20px) calc(100% - 10px) |||| calc 中间的-号两边要有空格

4.边框内圆角

border-radius 产生圆角 ，边框不是圆角，外层outline 设置后，圆角和边框间有空袭，用box-shadow 填充，它的大小设置为圆角的一般比较好

border-radius: .8em;
box-shadow: 0 0 0 0.4em pink;
outline: .6em solid pink;

5.条纹背景

横向条纹

background: linear-gradient(#ffeaf5 33.3%, #e7b0ff 0 , #e7b0ff 66.6%, #fff8cf 0);
background-size: 100% 45px;

liner-gradient 本来是有渐变的，但是上面这样设置就没有渐变了

background: linear-gradient(red, green 40px, orange 80px); red 向40px 处绿色渐变，然后向80px处橙色渐变

垂直条纹

background: linear-gradient(to right, #ffeaf5 50%, #e7b0ff 0);
background-size: 30px 100%;

to right ，从左向右渐变

斜向条纹

background: repeating-linear-gradient(45deg, #e7b0ff 0, #e7b0ff 15px, #ffeaf5 0,#ffeaf5 30px);

repeating-linear-gradient 重复渐变

灵活的同色系条纹

border: 1em solid transparent;
background: #ff1318;
background-image: repeating-linear-gradient(30deg, hsla(0,0%,100%,.3), hsla(0,0%,100%,.3) 15px, transparent 0 , transparent 30px);
